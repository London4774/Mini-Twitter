# Mini Twitter

Учебный pet-проект — мини-клон Twitter, реализованный втроём.

## Стек

**Frontend**
- React
- TailwindCSS
- Zustand
- React Router
- Axios

**Backend**
- Node.js
- Express
- PostgreSQL (pg)
- bcrypt
- jsonwebtoken
- dotenv
- cors

**Infrastructure**
- Docker
- AWS EC2
- AWS RDS
- Nginx
- CI/CD — GitHub Actions

## Запуск

**Frontend**
```bash
cd frontend
npm install
npm run dev
```

**Backend**
```bash
cd backend
npm install
npm run dev
```
